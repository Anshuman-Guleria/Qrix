# Qrix

A modern, privacy-first QR Code Generator built with React, TypeScript, and Vite.

Qrix enables users to create beautiful, customizable QR codes with support for multiple QR formats, logos, gradients, and high-quality exports—all processed entirely in the browser without requiring a backend or cloud services.

---

## Features

- Generate QR codes instantly
- Support for multiple QR types
  - URL
  - Text
  - Email
  - Phone
  - SMS
  - WiFi
  - Location
  - vCard
- Custom colors and gradients
- Logo embedding
- High-resolution PNG export
- SVG export
- PDF export
- Responsive modern UI
- Client-side processing
- Privacy-first architecture
- No authentication required
- No database
- No backend
- No Firebase
- No paid APIs

## Installation

Clone the repository

```bash
git clone https://github.com/Anshuman-Guleria/Qrix.git
```

Go to the project directory

```bash
cd Qrix
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```

---

## Privacy

Qrix is designed with privacy in mind.

- QR generation happens entirely inside the browser.
- No user data is stored.
- No tracking services.
- No Firebase.
- No cloud database.
- No server-side processing.
